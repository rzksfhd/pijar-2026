# PIJAR International Summer Course 2026

**Program for Innovation and Joint Action for Resilience**

## About

PIJAR 2026 is an International Summer Course on Sustainable Development Goals (SDGs) hosted by Universitas Terbuka in Indonesia.

### Theme
"Beyond Distance: Connecting Global Minds for Sustainable Development"

## Program Details

- **Location**: Jakarta & Bandung, Indonesia
- **Duration**: 8 Days Intensive Program
- **Learning Hours**: 45 hours of experiential learning
- **Participants**: Maximum 25 students

## Features

- Interactive lectures & workshops on SDGs
- Field visits to community sustainability initiatives
- Cross-cultural collaboration with international students
- Indonesian cultural heritage immersion
- Group project & final presentation

## Funding Coverage

- Meals during program
- Accommodation
- Program fees

## Eligibility

- International students (Undergraduate - Doctoral)
- Interest in Sustainable Development, Global Citizenship, and Cultural Exchange
